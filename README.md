# Hatemecha Web

Portfolio web construido con React, TypeScript y Vite.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Estructura

- `src/components`: piezas visuales reutilizables.
- `src/data`: contenido y rutas de assets.
- `src/hooks`: comportamiento de animacion, medicion y controles.
- `public/assets`: imagenes usadas por la app.
- `docs/brand-spec.md`: referencia de marca y layout.
